# Support

Ralph Loop の使い方や運用相談は、内容に応じて窓口を分けています。

## まず見る場所

- 導入と基本操作: [README.md](README.md)
- 最小構成のサンプル: [examples/minimal/README.md](examples/minimal/README.md)
- リリース手順: [docs/releasing.md](docs/releasing.md)
- セキュリティ報告: [SECURITY.md](SECURITY.md)

## 問い合わせの使い分け

- バグ報告: GitHub Issues の `Bug Report`
- 機能提案: GitHub Issues の `Feature Request`
- セキュリティ問題: 公開 issue ではなく `SECURITY.md`

## issue を開く前の確認

- `npm run check`
- `npm run test`
- `npm run build`
- `.env` の relevant な設定値
- 再現手順、期待結果、実際の結果

再現条件が揃っているほど、対応は速くなります。
