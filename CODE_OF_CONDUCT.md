# Code of Conduct

このリポジトリは、実務で使える品質の議論と改善を歓迎します。そのために、以下を最低限の行動基準とします。

## 期待するふるまい

- 事実と再現手順を優先して話す
- 人ではなく変更内容と挙動をレビューする
- 初心者でも追える言葉で補足する
- セキュリティや運用リスクは早めに共有する
- 反対意見があるときは、代案と根拠を添える

## 許容しないふるまい

- 個人攻撃、侮辱、威圧、差別的な表現
- 意図的な荒らし、スパム、執拗な連投
- 秘密情報、個人情報、認証情報の無断共有
- 再現確認や根拠なしに相手を断定的に非難すること

## 運用方針

- 明確な違反には、会話の停止、issue / PR のロック、ブロックなどで対応します
- 悪意ではなくても、継続的に運用を妨げるふるまいには是正を求めます
- セキュリティ事故や機密情報の混入は、公開 issue ではなく `SECURITY.md` の手順で報告してください

## 連絡先

困りごとや行動規範に関する相談は、公開 issue ではなく `SUPPORT.md` に記載の窓口から連絡してください。
